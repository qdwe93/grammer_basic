# 영어 문법 학습 앱 — 개발 문서

영어 초급 문법(문장성분·품사) 학습용 퀴즈 앱을 만들기 위한 개발 문서 모음입니다.
`resource_docs/`의 워크북(문제/정답/원문/해석)과 `design.md`(UI 디자인 시스템)를 기반으로 작성되었습니다.

## 이 앱이 하는 일 (한 줄 요약)

영어 문장을 단어 단위로 쪼개어 제시하고, 학습자가 각 단어의 **품사**와 각 구(phrase)의 **문장성분**을 맞히면, 채점·재시도·정답 공개(색상 코딩 + 한글 해석)·완료 축하로 이어지는 자기주도 학습 도구.

## 사용 방법

1. 먼저 [01_제품기획.md](01_제품기획.md) → [02_데이터모델.md](02_데이터모델.md) → [04_디자인적용가이드.md](04_디자인적용가이드.md)를 훑어 전체 그림을 파악합니다.
2. 실제 개발을 Claude Code에 맡길 때는 [06_개발프롬프트.md](06_개발프롬프트.md)의 **Phase 0**부터 순서대로 프롬프트를 복사해 입력합니다.
3. 각 Phase 프롬프트는 위 01~04 문서를 "참고 문서"로 가리키므로, 작업 디렉터리에 `docs/`가 함께 있어야 합니다.

## 핵심 설계 원칙 (전체 문서 관통)

- **2채널 색상 시스템**: 품사 = 단어 배경 **틴트**, 문장성분 = 구 아래 **밑줄 브래킷 + 라벨 칩**. 두 채널은 시각적으로 절대 섞이지 않는다.
- **색은 보상이다**: 답하는 동안 보드는 무채색. 정답을 모두 맞혀야 색상·해석이 공개된다.
- **데이터 소스**: 저장된 JSON 시드 스토리(앱에 내장).
- **개인용**: 백엔드 없음. 학습 기록은 브라우저 로컬에만 저장.

> `ignore_this/` 폴더는 무시합니다.
