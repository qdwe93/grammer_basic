# 06. 개발 프롬프트 모음 (Claude Code용)

이 문서의 각 Phase 프롬프트를 **순서대로 복사해 Claude Code에 입력**하면 앱이 단계적으로 완성된다.
모든 프롬프트는 같은 작업 디렉터리(`docs/`가 함께 있는 곳)에서 실행한다고 가정한다.

## 사용 규칙
- 한 번에 한 Phase씩 입력하고, 결과를 확인한 뒤 다음으로 넘어간다.
- 프롬프트 안의 "참고 문서"는 반드시 읽게 한다(잘못된 추측 방지).
- 색·라벨·스키마는 **추측 금지, 문서 값 그대로** 사용하도록 매 프롬프트에 못 박는다.
- `ignore_this/`는 항상 무시.
